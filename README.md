# پوسته Skyler برای Typecho

![Skyler Theme Preview](https://raw.githubusercontent.com/abdulhalim/skyler/refs/heads/main/screenshot.png)


**پوسته اسکایلر (Skyler)** یک پوسته مدرن، سریع و منعطف برای سیستم مدیریت محتوای Typecho است که برای وبلاگ‌ها و سایت‌های محتوایی فارسی طراحی شده است.

## امکانات و ویژگی‌ها

* طراحی کاملاً ریسپانسیو
* منوی پویا با قابلیت انتخاب دسته و صفحه
* ویجت‌های قابل تنظیم در سایدبار
* فیلدهای سفارشی نوشته:

  * بنر
  * لینک دانلود
  * لینک پیش‌نمایش
  * گالری تصاویر
* گالری تصاویر با لایت‌باکس
* دکمه کپی خودکار کد
* دکمه بازگشت به بالا
* باکس نویسنده با آواتار و شبکه‌های اجتماعی
* سیستم نظرات بازگشتی (تا سطح ۲)
* صفحه 404 اختصاصی
* صفحه لینک‌تری (Linktree) مستقل
* صفحه درباره من اختصاصی
* بهینه‌سازی سئو
* پشتیبانی از تاریخ شمسی (با افزونه jDate)
* نمایش تعداد بازدید (با افزونه ViewsCounter)
* فونت Vazirmatn و FontAwesome به‌صورت آفلاین

## نصب و راه‌اندازی

1. پوشه قالب را در مسیر زیر آپلود کنید:

```
/usr/themes/skyler/
```

2. از پیشخوان مدیریت:

```
ظاهر → مدیریت پوسته‌ها → Skyler
```

را فعال کنید.

3. سپس از مسیر زیر تنظیمات پوسته را انجام دهید:

```
ظاهر → تنظیمات پوسته
```

> فونت‌ها و آیکون‌ها به صورت آفلاین داخل قالب قرار داده شده‌اند.
> در صورت تغییر مسیر پوشه‌ها، مسیرها را در فایل `header.php` اصلاح کنید.

## تنظیمات پوسته

پس از فعال‌سازی قالب، بخش «تنظیمات پوسته» در منوی ظاهر اضافه می‌شود.

### تنظیمات عمومی

| گزینه               | توضیح                                               |       |
| ------------------- | --------------------------------------------------- | ----- |
| آدرس لوگو سایت      | لینک تصویر لوگو (در صورت خالی بودن: `img/logo.png`) |       |
| آدرس فاویکون        | پیش‌فرض: `img/favicon.ico`                          |       |
| لینک‌های سفارشی منو | هر خط به صورت `عنوان                                | آدرس` |
| آدرس آواتار نویسنده | در صورت خالی بودن از آواتار پیش‌فرض استفاده می‌شود  |       |
| درباره نویسنده      | متن کوتاه برای سایدبار و باکس نویسنده               |       |

---

### تنظیمات شبکه‌های اجتماعی

لینک شبکه‌های اجتماعی در این بخش وارد می‌شود و در قسمت‌های زیر استفاده می‌شود:

* صفحه درباره من
* صفحه لینک‌تری
* باکس نویسنده
* فوتر سایت

---

### تنظیمات ویجت‌های سایدبار

* جستجو
* درباره نویسنده
* عنوان ویجت درباره نویسنده
* پست‌های اخیر
* نظرات اخیر
* دسته‌بندی‌ها
* برچسب‌ها

> با فعال‌سازی «استایل ویژه درباره نویسنده»، این ویجت با رنگ اصلی قالب نمایش داده می‌شود.

---

### تنظیمات منوی بالا

* انتخاب دسته‌هایی که کنار لینک خانه نمایش داده شوند
* انتخاب صفحات مستقل برای منوی بالا

---

### تنظیمات فوتر

* انتخاب صفحاتی که به‌صورت لینک در فوتر نمایش داده می‌شوند

## فیلدهای سفارشی نوشته‌ها

در انتهای ویرایشگر نوشته‌ها، بخش «فیلدهای سفارشی نوشته» اضافه می‌شود.

| نام فیلد              | کاربرد                       |
| --------------------- | ---------------------------- |
| آدرس تصویر شاخص (بنر) | تصویر بزرگ بالای نوشته       |
| لینک دانلود فایل      | نمایش دکمه دانلود            |
| لینک پیش‌نمایش        | نمایش دکمه پیش‌نمایش         |
| لینک‌های گالری تصاویر | هر لینک تصویر در یک خط       |
| استایل گالری          | `style-zoom` یا `style-dark` |
| تعداد ستون‌ها         | ۲، ۳ یا ۴ ستون               |
| نسبت تصویر            | افقی، عمودی یا مربع          |

نمونه ورودی گالری:

```
https://example.com/image1.jpg
https://example.com/image2.jpg
https://example.com/image3.jpg
```

## شخصی‌سازی دستی (هاردکدینگ)

> این بخش نیازمند ویرایش مستقیم فایل‌هاست. قبل از هر تغییری بکاپ بگیرید.

| فایل         | توضیح                                    |
| ------------ | ---------------------------------------- |
| `about.php`  | ویرایش نام و اطلاعات صفحه درباره من      |
| `links.php`  | عنوان و توضیحات صفحه لینک‌تری            |
| `header.php` | لینک و متن دکمه CTA و زیرعنوان صفحه اصلی |
| `footer.php` | متن کپی‌رایت                             |
| `index.php`  | عنوان اصلی صفحه خانه                     |

## راهنمای بخش‌ها

### گالری تصاویر

* لینک تصاویر را در فیلد مربوطه وارد کنید.
* نوع نمایش، تعداد ستون و نسبت تصویر قابل انتخاب است.
* گالری با لایت‌باکس نمایش داده می‌شود.

---

### دکمه کپی کد

برای تمام بلوک‌های:

```
<pre><code>
```

دکمه «کپی کد» اضافه می‌شود.

---

### دکمه بازگشت به بالا

پس از اسکرول بیش از ۳۰۰ پیکسل نمایش داده می‌شود.

---

### سیستم نظرات

* امکان پاسخ‌دهی تا ۲ سطح
* اسکرول خودکار فرم پاسخ
* اولویت آواتار مدیر:

  1. فیلد `authorAvatar`
  2. تصویر پیش‌فرض
  3. گراواتار

## عیب‌یابی

| مشکل                             | راه‌حل                                                                 |
| -------------------------------- | ---------------------------------------------------------------------- |
| فونت یا آیکون لود نمی‌شود        | وجود فایل‌های `fonts.css`، `all.css` و پوشه `webfonts` را بررسی کنید   |
| تاریخ میلادی است                 | افزونه jDate را فعال کنید                                              |
| تعداد بازدید نمایش داده نمی‌شود  | افزونه ViewsCounter باید فعال باشد                                     |
| تصویر شاخص نمایش داده نمی‌شود    | اولویت: فیلد banner → اولین تصویر محتوا → تصویر پیش‌فرض → تصویر تصادفی |
| منو دسته یا صفحه را نشان نمی‌دهد | تنظیمات منو در تنظیمات پوسته بررسی شود                                 |
| پاسخ به کامنت کار نمی‌کند        | بارگذاری `comments.js` و مقداردهی `respondId` بررسی شود                |
